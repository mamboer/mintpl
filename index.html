Installation

Install manually

Add the mintpl.js to your page:

<script src="/path/to/mintpl.min.js"></script>

Or, load the mintpl.js using your AMD loader like ```requirejs```

define(['mintpl'],function(mintpl){/* module logic goes here */});

Install via Bower

bower install mintpl --save

API

heredoc

heredoc let you define readable inline html templates inside javascript
new mintpl.heredoc(function(){/*!
    
<%=name%>

*/});
Run

render

mintpl.render( tplId | heredoc | heredoc function, data );

You can:


 var html = mintpl.render(function(){/*!
    

        
<%=name%>

        
Author: <%=author%>

    

*/},{name:'mintpl',author:'Levin Van <github.com/mamboer>'});
var dom = document.createElement('div');
dom.className = 'test';
dom.innerHTML = html;
document.getElementById('test').appendChild(dom);
Run

Or, just pass a template id:


 <script type="text/mintpl" id="testTpl" />
    

        
<%=name%>

        
Author: <%=author%>

    

</script>
                    

var html = mintpl.render('testTpl', {name:'mintpl',author:'Levin Van <github.com/mamboer>'});
var dom = document.createElement('div');
dom.className = 'test';
dom.innerHTML = html;
document.getElementById('test').appendChild(dom);
Run

Build

To build `mintpl` from sources you need to install Gulp 

$ npm install gulp -g

Then navigate in terminal to the downloaded sources and install some required modules 

$ npm install gulp gulp-uglify gulp-swig gulp-rename

Then run Gulp 

$ gulp

You can take compressed and optimized files from dist folder
